 AsiCoffee

Projeto desenvolvido para a Semana 3 da capacitação de Flutter.

## Objetivo

Criar a interface visual de um aplicativo de cafeteria usando Flutter, praticando widgets, layout, componentização e estado local.

## Funcionalidades

- Tela principal com AppBar personalizada.
- Lista de produtos do cardápio.
- Imagens carregadas por URL.
- Nome, categoria e preço dos produtos.
- Botão de favorito com alteração visual ao clicar.
- Uso do pacote google_fonts.
- Componentização em arquivos separados.
